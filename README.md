# overloading

Tony's repository — 一些有趣的 HTML 小游戏合集，打开即玩，无需安装。

## 🎮 游戏列表

| 游戏 | 文件 | 简介 |
|------|------|------|
| 🏁 禅意五子棋 | [html/gomoku.html](html/gomoku.html) | 经典五子棋，水墨禅意风格，双人对弈 |
| 🃏 24点 | [html/make24.html](html/make24.html) | 经典24点纸牌游戏，用四则运算凑出24 |
| 🐍 贪吃蛇 | [html/snake-game.html](html/snake-game.html) | 经典贪吃蛇，支持键盘和触屏操控 |

## 🚀 使用方式

直接在浏览器中打开 `html/` 目录下的 `.html` 文件即可开始游戏，无需任何依赖或服务器。

## 📁 项目结构

```
overloading/
├── html/
│   ├── gomoku.html        # 五子棋
│   ├── make24.html        # 24点
│   └── snake-game.html    # 贪吃蛇
├── .gitignore
└── README.md
```
