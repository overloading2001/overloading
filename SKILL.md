# 妇产科护理反思日记自动评改

## 一句话描述

自动批量评改妇产科护理专业学生的反思日记，**内置 AI 创作识别机制（ai-detector / GPTHumanizer API）**，严格依据《妇产科护理学》反思日记评价量规（2026年修订版），生成逐份评改报告（Markdown）和班级成绩汇总表（Excel）。

---

## AI 创作识别规则（⚠️ 核心功能）

系统通过 **ai-detector skill（GPTHumanizer API）** 检测 AI 生成特征，**AI 概率 > 40% 直接判定不合格**，并在报告和汇总表中标注警示。

> 已优化：降低误判率，正常学生真实写作不会被标记为 AI。

### 检测引擎

| 优先级 | 引擎 | 说明 |
|--------|------|----------|
| 第一优先 | `ai-detector` API | 调用 GPTHumanizer API，返回 `human/ai/ai_humanized/light_edited` 分类 + AI 概率（0-1） |
| 回退 | 内置规则检测 | API 调用失败时自动回退，检测句式均匀度/AI套话/过渡词/个人化表达/内容泛化度/段落整齐度 |

### API 说明

- **端点**：`https://detect.gpthumanizer.ai/api/detect_ai`
- **方法**：POST
- **认证**：无需认证
- **速率限制**：100次/分钟/IP，10,000次/天/IP
- **返回字段**：
  - `class`：`human` / `ai` / `ai_humanized` / `light_edited`
  - `ai_possibilities`：AI 可能性（0-1，转换为 0-100% 显示）
  - `probabilities`：各类别概率分布

### 处置规则

- **AI 概率 > 40%**（API 返回 `ai_possibilities` > 0.4）→ 总分强制置 0，等级标注为 `不合格（AI创作）`，Markdown 报告顶部显示红色警示区块
- **AI 概率 ≤ 40%** → 正常评分，不受影响

### 典型 AI 套话库（部分示例）

```
"综上所述"、"总而言之"、"值得注意的是"、"需要指出的是"、
"首先...其次...最后..."、"一方面...另一方面..."、
"具有十分重要的意义"、"为...奠定了坚实基础"
```

## 使用方法

**当用户说"帮我评改反思日记"/"批量评改"/"自动批改日记"时，执行以下步骤：**

### 第一步：让用户指定文件夹

询问用户存放反思日记的文件夹路径。支持格式：
- `.docx`（Word 文档）
- `.txt`（纯文本）

### 第二步：运行评改脚本

```bash
python "C:\Users\Administrator.DESKTOP-FEUGF3U\.workbuddy\skills\nursing-reflection-grader\scripts\grader.py" "<文件夹路径>"
```

可选参数 `--output` 指定输出目录（默认在输入文件夹下创建"评改结果"子文件夹）：

```bash
python ".../grader.py" "C:\反思日记" --output "C:\评改输出"
```

### 第三步：查看结果

脚本运行完成后，输出目录中会生成：
- **每位学生的 Markdown 评改报告**（含8维度评分 + 评语 + 参考标准 + AI 检测详情）
- **班级成绩汇总.xlsx**（含成绩排列 + 维度分析 sheet）

## 评分标准（来源：《妇产科护理学》反思日记评价量规）

**满分 100 分，8 个维度**

| 序号 | 评价维度 | 分值 | 评价重点 |
|------|----------|------|----------|
| 1 | 情境呈现与问题意识 | 10分 | 能否从孕产妇、胎儿/新生儿、家庭和护理团队多视角识别问题 |
| 2 | 生命至上与母婴安全意识 | 15分 | 是否体现风险意识、急危重症识别、母婴安全优先 |
| 3 | 课程价值与职业认知 | 15分 | 是否理解妇产科护士职业使命、专业责任、团队协作和护理价值 |
| 4 | 人文关怀与共情能力 | 15分 | 是否理解孕产妇焦虑、疼痛、羞耻感、丧失体验等情绪 |
| 5 | 专科护理思维与风险应对 | 15分 | 是否能结合妇产科护理知识分析问题并提出合理措施，体现专业判断、风险意识能力 |
| 6 | 科学精神与思辨思维 | 10分 | 是否能秉持严谨科学态度，思辨研判，循证反思优化护理举措 |
| 7 | 职业认同与责任担当 | 10分 | 是否体现慎独精神、团队协作、守护生命的职业使命 |
| 8 | 自我反思与改进计划 | 10分 | 能否反思自身不足，并提出具体可行的改进措施 |

**综合等级阈值（满分100）：**

| 等级 | 分数线 |
|------|--------|
| 优秀 | ≥ 90 分 |
| 良好 | ≥ 75 分 |
| 合格 | ≥ 60 分 |
| 不合格 | < 60 分 |

**各维度等级阈值：优秀 ≥ 90% / 良好 ≥ 75% / 合格 ≥ 60%**

> **评分方法**：采用语义优先评分——关注具体事件描述、多视角分析、关怀行为、专业操作，而非仅数关键词。各维度均配备专项语义评分函数。
